# Original Static Site

This folder contains the original static HTML/CSS/JS version of the portfolio before it was migrated to React.

## Files

- `index.html` - Original HTML structure with Bootstrap
- `index.css` - Original custom styles
- `sendMeFeedbackModal.js` - Original mailto functionality

## Migration

The site was migrated to:
- React 19 + TypeScript
- Vite (build tool)
- Ant Design (UI library)
- Storybook (component documentation)

See the main README.md for the new project structure.
